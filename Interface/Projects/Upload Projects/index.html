🔄 Set Up Bi-Directional Sync?

For seamless updates between Activity Hub and your source, we recommend setting up an API connection.

⏱️ Automatic sync on schedule (hourly, daily, weekly)
↔️ Changes in Activity Hub push back to your source
🆕 New projects from source automatically appear here
⚡ Conflict detection and resolution
⚠️

Sync Warning

Without an API connection, your data may become out of sync:

  • Changes made in your source won't appear in Activity Hub automatically
  • Changes made here won't update your original system
  • You'll need to manually re-upload to sync data

You can set up an API connection later in Settings → Data Connections.
