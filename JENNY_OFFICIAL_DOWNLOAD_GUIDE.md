# Jenny Voice Download & Installation Guide

## Official Source

**Jenny is NOT available as a standalone download.**

Jenny (and all Narrator voices) are distributed **exclusively through Microsoft Store/Windows Update**. There is no direct download link from Microsoft's website.

### Official Installation Path

1. **Settings → Accessibility → Text-to-speech → Manage voices**
2. Click **"Add voices"** button
3. Browse available voices (Jenny, Aria, Guy, Mark, etc.)
4. Click **"Get"** next to "Microsoft Jenny"
5. Wait for installation to complete (may take 2-5 minutes)
6. Restart computer

### Why This Matters

- Microsoft controls distribution to ensure compatibility
- Voices integrate with Windows Update for patches/fixes
- Direct downloads bypass quality assurance

---

## Alternative: Contact Microsoft Support

If Jenny is not appearing in "Add voices" list:
1. Open **Settings → System → Troubleshoot → Additional troubleshooting options**
2. Look for "Speech Synthesis" or "Text-to-speech" issues
3. Or contact Microsoft Support for voice installation issues

## Why Jenny Installation Failed Previously

Possible causes:
- Windows Update cache corruption
- Incomplete download/installation
- Registry mismatch between apps and voice packages
- Missing language pack dependencies

**Solution:** 
1. Uninstall completely (Settings → Apps → Voice)
2. Restart
3. Reinstall via Settings → Add voices
4. Restart again

No external download needed - it all comes from Microsoft's servers through Settings.
