# Landing Page - Activity Hub

## 📄 Overview
The Activity Hub Landing Page serves as the primary entry point for users accessing the enterprise dashboard system. This page showcases the key features, provides navigation to different sections, and demonstrates the integrated Sparky AI assistant.

## 🎯 Purpose
- **First Impression**: Professional introduction to Activity Hub
- **Feature Showcase**: Highlight key capabilities and widgets
- **User Onboarding**: Guide new users through the platform
- **Sparky AI Integration**: Demonstrate intelligent assistant capabilities
- **Brand Consistency**: Maintain Walmart corporate identity

## 📁 Files
- **`index.html`** - Main landing page (production version)
- **`activity-hub-demo.html`** - Original Activity Hub demo and prototype
- **Dependencies**: Links to `/Design/walmart-brand-variables.css`

## 🎨 Design Elements

### Visual Components
- **Hero Section**: Prominent title and value proposition
- **Feature Cards**: Key widget showcases (Dashboard, Tasks, Metrics, etc.)
- **Interactive Widgets**: Live demonstrations of functionality
- **Sparky AI Widget**: Integrated assistant with 😉 character
