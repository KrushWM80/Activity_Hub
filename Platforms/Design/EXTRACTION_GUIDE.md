# Walmart PPT Template Brand Extraction Guide

## Step-by-Step Instructions

### 1. Open the PowerPoint Template
Open `Walmart_PPT_Template_240225.pptx` in Microsoft PowerPoint.

### 2. Extract Color Information

#### Method A: Using PowerPoint's Design Tab
1. Go to **Design** tab in PowerPoint
2. Click on **Variants** dropdown
3. Select **Colors** → **Customize Colors**
4. This will show you the theme colors being used

#### Method B: Using Format Painter
1. Right-click on any Walmart blue element (logo, header, etc.)
2. Select **Format Shape** or **Format Text**
3. Go to **Fill** or **Font Color**
4. Click **More Colors** → **Custom** tab
5. Note down the RGB values or Hex codes

#### Method C: Using Developer Tools (if web-based)
1. If viewing online, right-click and **Inspect Element**
2. Look for CSS color values in the styles panel

### 3. Extract Typography Information

#### Font Family
1. Select any text element
2. Look at the **Font** dropdown in the Home tab
3. Note the primary font family used

#### Font Sizes
1. Select different text elements (titles, body text, etc.)
2. Note the font sizes shown in the **Font Size** dropdown
3. Document the hierarchy of sizes used

### 4. Extract Layout Measurements

#### Spacing and Margins
1. Go to **View** → **Guides** → **Grid and Guides**
2. Check spacing between elements
3. Use the ruler to measure margins and padding

#### Logo Specifications
1. Select the Walmart logo
2. Check its dimensions in the **Size** section
3. Note minimum clear space around the logo
