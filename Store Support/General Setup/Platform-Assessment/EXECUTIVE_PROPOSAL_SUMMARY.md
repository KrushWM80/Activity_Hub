## How It Works

### Input Phase
```
1. Complete assessment using assessment_tool.html
2. Note the key results:
   - Platform name
   - Complexity level
   - Costs (dev & annual)
   - Timeline
   - User count
   - Strategic objectives
   - Benefits
```

### Processing Phase
```
3. Open executive_proposal_generator.html
4. Either:
   a) Paste JSON from assessment, OR
   b) Manually fill the form
5. Click "Generate Proposal"
```

### Output Phase
```
6. Review generated proposal
7. Choose output format:
   - Print to PDF
   - Download as HTML
   - Share as link
```
