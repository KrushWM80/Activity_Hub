# Code Puppy Pages Development Resources

This folder contains development guides, utilities, and best practices for building applications for Walmart's Code Puppy Pages platform.

## Contents

- **CODE_PUPPY_GUIDE.md** - Comprehensive development guide
- **EMOJI_FIX_GUIDE.md** - UTF-8 emoji encoding troubleshooting
- **DATA_ARCHITECTURE.md** - Data structure and filtering patterns
- **fix_emoji.py** - Python utility for fixing broken emoji
- **fix-unicode.js** - JavaScript utility for unicode fixes
- **fix-emoji.js** - Additional emoji fix script

## Quick Start

Code Puppy Pages (https://puppy.walmart.com) is a static hosting platform that requires:
- Single-file HTML deployment
- All CSS, JS, and data embedded
- UTF-8 encoding for emoji support
- Pre-calculated aggregations for performance

See the individual guides for detailed information.
