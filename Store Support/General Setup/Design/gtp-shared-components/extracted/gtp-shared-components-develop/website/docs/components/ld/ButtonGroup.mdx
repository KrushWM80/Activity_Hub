Button group displays multiple related actions in a horizontal row
to help with arrangement and spacing.


<ScreenshotTabs screenshotData={"screenshots/button-group.png"} baseUrl="/pages/electrode-mobile-platform/gtp-shared-components/"/>
## Usage
```js
import {ButtonGroup} from '@walmart/gtp-shared-components`;

<ButtonGroup>
  <Button
    variant="secondary"
    onPress={() => {}}>
    Secondary
  </Button>
  <Button
    variant="primary"
    onPress={() => {}}>
    Primary
  </Button>
</ButtonGroup>
```
