Popovers are overlays that display contextual content on demand.


<ScreenshotTabs screenshotData={"screenshots/popover.png"} baseUrl="/pages/electrode-mobile-platform/gtp-shared-components/"/>
## Usage
```js
import {Popover} from '@walmart/gtp-shared-components`;

const [isOpen, setIsOpen] = useState(false);

<Popover
 content={'Example Popover content.'}
 isOpen={isOpen}
 onClose={() => setIsOpen(false)}
 position="topLeft">
  <Link
    onPress={() => setIsOpen(!isOpen)}>
    Top{'\n'}Left
  </Link>
</Popover>
```
