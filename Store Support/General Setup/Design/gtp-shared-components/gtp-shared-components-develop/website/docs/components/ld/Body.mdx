## Usage
```js
import {Body} from '@walmart/gtp-shared-components`;

<Body>Lorem ipsum dolor sit amet</Body>
<Body size="small" weight="400">Lorem ipsum dolor sit amet</Body>
<Body isMonospace>Lorem ipsum dolor sit amet</Body>
```
