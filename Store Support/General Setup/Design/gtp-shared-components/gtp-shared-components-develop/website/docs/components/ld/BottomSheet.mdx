Bottom Sheets are surfaces anchored to the bottom of the screen that contain supplementary content.


<ScreenshotTabs screenshotData={{"small":"screenshots/bottom-sheet-small.png","large":"screenshots/bottom-sheet-large.png"}} baseUrl="/pages/electrode-mobile-platform/gtp-shared-components/"/>
## Usage
```js
import {BottomSheet} from '@walmart/gtp-shared-components`;

const [modalIsOpen, setModalIsOpen] = React.useState(false);

<BottomSheet
  isOpen={modalIsOpen}
  onOpen={() => setModalIsOpen(true)}
  onClose={() => setModalIsOpen(false)}
  title="Confirmation"
  actions={
    <ButtonGroup>
      <Button variant="tertiary" onPress={() => setModalIsOpen(false)}>
        Cancel
      </Button>
      <Button variant="primary" onPress={() => setModalIsOpen(false)}>
        Continue
      </Button>
    </ButtonGroup>
  }>
  Lorem ipsum dolor sit amet
</BottomSheet>
```
