## Key Information

### Three-Format System
```
SMART Code: 1-993-1026 (human-readable)
           ↓
Workday Code: US-01-0202-002104 (structured)
           ↓
User ID: e0c0l5x.s03935 (system reference)
```

### Main Data Sources
1. **job_codes_master.json** (44,934 lines) - Local master bridge
2. **Polaris** (BigQuery) - Current employee assignments
3. **CoreHR** (BigQuery) - Master employee profiles
4. **Local BQ** (wmt-assetprotection-prod) - AMP-specific data
