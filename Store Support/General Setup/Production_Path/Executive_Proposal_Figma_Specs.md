## Creating in Figma - Step by Step

### Step 1: Set Up File
1. Create new Figma file
2. Set canvas to 1920x1080px
3. Create 20 frames (one per slide)
4. Name frames: "Slide 01 - Title", "Slide 02 - Executive Summary", etc.

### Step 2: Build Slides
1. Start with Slide 1 (Title)
2. Use specifications above for layout
3. Add text, shapes, icons per slide spec
4. Apply colors from palette reference
5. Repeat for all 20 slides

### Step 3: Add Animations (Figma Prototype)
1. Select frame
2. Click "Prototype" tab
3. Set interactions (After delay → Navigate to next frame)
4. Add element animations (Smart Animate between frames)
5. Set timing per specifications above

### Step 4: Export Video
1. Select all frames in order
2. Click "Play" button (prototype mode)
3. Use screen recording or Figma's export feature
4. Export as MP4 with settings specified above
