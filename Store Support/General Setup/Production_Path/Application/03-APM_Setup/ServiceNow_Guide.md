# ServiceNow: Application Hub (APM)

## 🔗 Application Hub Portal

APM registration and all assessments are completed via ServiceNow Application Hub:

- **Application Hub**: https://walmartglobal.service-now.com/apm?id=wm_application_hub
- **APM Onboarding Request**: https://walmartglobal.service-now.com/wm_sp/?id=sc_cat_item_guide&sys_id=e5a706c887c3cd102d2433fc3fbb35c3
- **Access**: Requires Walmart authentication

---

## 📋 Using ServiceNow Application Hub

### Initial APM Registration:

1. **Access Application Hub**
   - [ ] Go to https://walmartglobal.service-now.com/apm?id=wm_application_hub
   - [ ] Log in with Walmart credentials
   - [ ] Verify Application Owner has no pending certifications

2. **Submit Onboarding Request**
   - [ ] Click "Register Application" or access onboarding catalog item
   - [ ] Complete "Name" and "Description" fields (business-focused!)
   - [ ] Fill all required fields
   - [ ] Submit request
   - [ ] Record your APM# when assigned

3. **Navigate "My Pending Assessments"**
   - [ ] Locate "My Pending Assessments" in center of Application Hub
   - [ ] This is where all assessments (DCA, PCI, RISK, EPRA) will appear
   - [ ] Track completion status for each assessment

---

## 📊 ServiceNow Assessment Workflow

### Data Classification Assessment (DCA):

1. **Access DCA**
   - [ ] Go to Application Hub
   - [ ] Open "My Pending Assessments"
   - [ ] Locate and click "Data Classification Assessment"

2. **Complete DCA**
   - [ ] Answer all questions about data types
   - [ ] Select applicable categories (PCI, FINC, SOX) - **PERMANENT choices!**
   - [ ] Provide business justification for each category
   - [ ] Submit assessment

3. **Track Auto-Triggered Assessments**
   - [ ] Return to "My Pending Assessments"
   - [ ] Complete PCI Assessment (if PCI data)
   - [ ] Complete RISK Assessment (always required)
   - [ ] Complete EPRA via OneTrust (if triggered)

## 📝 ServiceNow APM Best Practices

### Navigation Tips:
- **Bookmark Application Hub** for quick access
- **Check "My Pending Assessments" daily** - Don't let tasks sit idle
- **Save screenshots** of completed assessments
- **Document your APM#** immediately upon receipt

### Common ServiceNow Actions:
- **View APM Record**: Search for your application in Application Hub
- **Check Certification Status**: View in your application's record
- **Update Information**: Edit application record as needed
- **Track Progress**: Monitor assessment completion dates

### Troubleshooting:
- **Can't find assessment**: Refresh Application Hub, check filters
- **Assessment not triggered**: Verify APM record is approved and active
- **Access denied**: Verify you're listed as Application Owner or Technical Owner
- **Stuck assessment**: Contact APM support: apmmailbox@wal-mart.com

## ⚠️ Critical ServiceNow Notes

### Before Submitting:
- ✅ Verify Application Owner has **no pending certification tasks**
- ✅ Draft descriptions **outside ServiceNow first** for review
- ✅ Have all required information ready before starting

### During Assessments:
- ✅ Complete assessments in order (DCA first, then auto-triggered)
- ✅ Don't skip questions - incomplete assessments delay certification
- ✅ **DCA data classifications are PERMANENT** - choose carefully!

### After Certification:
- ✅ Save certification confirmation
- ✅ Document APM# for SSP process (Step 4)
- ✅ Keep Application Hub bookmarked for future updates
