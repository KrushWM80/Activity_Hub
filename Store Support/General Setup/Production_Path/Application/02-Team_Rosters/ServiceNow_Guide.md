# ServiceNow: Team Rosters Requests

## 🔗 ServiceNow Portal Access

Team Rosters Product ID requests are submitted through ServiceNow:

- **Main Portal**: https://walmartglobal.service-now.com/
- **Team Rosters Portal**: Access via ServiceNow navigation
- **Access**: Requires Walmart authentication

---

## 📋 Using ServiceNow for Team Rosters

### Step-by-Step Process:

1. **Access ServiceNow**
   - [ ] Go to https://walmartglobal.service-now.com/
   - [ ] Log in with your Walmart credentials
   - [ ] Ensure you have proper permissions

2. **Navigate to Team Rosters**
   - [ ] Your DQC will access Team Rosters admin functions
   - [ ] Navigate: **Admin tab** → **Request Product Teams** → **Request Product Team**

3. **Submit Request**
   - [ ] DQC completes all required fields with information you provide
   - [ ] DQC submits request for approval
   - [ ] Note the request ticket number (RITM#)

4. **Track Request**
   - [ ] Monitor request status in ServiceNow
   - [ ] Check "My Requests" or "My Items"
   - [ ] Await Product ID assignment notification

---

## 📝 ServiceNow Best Practices

### Documentation:
- **Save your RITM#** (Request Item ticket number)
- Document submission date
- Track approval workflow
- Record Product ID when received

### Troubleshooting:
- If access issues: Contact IT Service Desk
- For status updates: Check ServiceNow "My Requests"
- For approval delays: Contact your DQC
