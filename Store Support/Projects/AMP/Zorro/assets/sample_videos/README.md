# Sample Videos for Demo Mode

Place your sample video here for leadership demos.

## Expected file:
- **walmart_associate_stocking.mp4** - Video of associate stocking shelves

## How to get a sample video:

### Option 1: Use existing Walmart media
Download a 5-8 second clip from Walmart Media Library or internal training videos.

### Option 2: Stock video sites (for internal demo only)
- Pexels.com (free, no attribution required)
- Search: "retail worker stocking shelves" or "supermarket employee"

### Option 3: Record your own
Record a quick 5-8 second clip on your phone in a Walmart store.

## File requirements:
- Format: MP4
- Duration: 5-8 seconds
- Resolution: 1080p preferred
- File name: walmart_associate_stocking.mp4

Once placed here, the demo mode will automatically use this video!
