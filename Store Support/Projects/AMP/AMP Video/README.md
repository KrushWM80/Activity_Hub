# AMP Video Creation Workflow

## 🎬 Video Creation Setup for AMP Content

This directory contains tools and scripts for creating AMP training videos.

### 📁 Directory Structure:
- `scripts/` - Video scripts and storyboards
- `assets/` - Images, logos, and visual elements  
- `audio/` - Voice recordings and background music
- `output/` - Final video files
- `templates/` - Video templates and layouts

### 🛠️ Tools Needed:

#### Option 1: Professional Tools
- **Adobe Premiere Pro** - Professional video editing
- **Adobe After Effects** - Motion graphics and animations
- **Audacity** - Audio editing (free)

#### Option 2: Free/Budget Tools
- **DaVinci Resolve** - Professional-grade free video editor
- **OpenShot** - Simple, free video editor
- **Canva** - Online video creation with templates
- **OBS Studio** - Screen recording and basic editing

#### Option 3: AI-Powered Tools
- **Synthesia** - AI avatar videos
- **Loom** - Quick screen recordings with narration
- **InVideo** - AI-powered video creation
- **Pictory** - Text-to-video conversion

### 🎯 30-Second Video Structure:
1. **0-5s**: Hook/Opening statement
2. **5-20s**: Main content/key points  
3. **20-25s**: Action/implementation
4. **25-30s**: Call to action/closing

### 📝 Content Extraction Process:
1. Access the AMP preview link
2. Extract key information and main points
3. Create script with visual cues
4. Record voiceover or use text-to-speech
5. Add visuals, graphics, and branding
6. Edit and export final video

### 🎨 AMP Branding Guidelines:
- Use Walmart corporate colors (blue #004c91, yellow #ffc220)
- Include Walmart spark logo
- Professional, clear typography
- Consistent with AMP dashboard styling
