# 📸 HOW TO CREATE IMAGE FOR WORKVIVO

## ⚡ QUICKEST METHOD (30 seconds)

### Using Windows Snipping Tool:
1. Open `workvivo_social_post.html` in your browser (Chrome/Edge)
2. Press **Windows Key + Shift + S** on your keyboard
3. Click and drag to select the entire white post area (1080x1920)
4. The screenshot is now in your clipboard
5. Open Paint or paste directly into an image editor
6. Save as `workvivo_winter_health.png`
7. Upload to WorkVivo ✅

## 🌐 BROWSER METHOD (Chrome/Edge)

### Full Page Screenshot:
1. Open `workvivo_social_post.html` in Chrome or Edge
2. Press **F12** to open Developer Tools
3. Press **Ctrl + Shift + P** to open Command Menu
4. Type "screenshot" and select **"Capture node screenshot"**
5. Click on the white `.social-post` element
6. Image will automatically download as PNG
7. Upload to WorkVivo ✅

## 🖱️ RIGHT-CLICK METHOD

### If you can see the full post on screen:
1. Right-click anywhere on the white post
2. Look for "Save image as..." or "Take screenshot"
3. Save the file
4. Upload to WorkVivo ✅

## 🔧 AUTOMATED METHOD (if you have Python packages)

### Install required packages:
```powershell
pip install selenium
pip install webdriver-manager
```

### Run the script:
```powershell
python create_workvivo_image.py
```

This will automatically generate `workvivo_winter_health.png` ready for upload!

## 📱 FINAL IMAGE SPECS
- **Size**: 1080 x 1920 pixels (9:16 ratio)
- **Format**: PNG (recommended) or JPG
- **File size**: Should be under 5MB
- **Platform**: Optimized for WorkVivo mobile feed

## ✅ VERIFICATION
After creating the image:
- ✓ Check dimensions are 1080 x 1920
- ✓ All content is visible (not cut off)
- ✓ Text is crisp and readable
- ✓ Colors look vibrant (Walmart blue and yellow)
- ✓ File size is reasonable for upload
