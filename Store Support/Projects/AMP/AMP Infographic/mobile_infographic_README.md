## Usage Instructions

### Viewing the Infographics
1. Open either HTML file in any modern web browser
2. View on mobile device or resize browser window to mobile width
3. Files work offline - no internet connection required after download

### Sharing Options
- **Web**: Upload to internal servers or SharePoint
- **Email**: Attach HTML files directly
- **Social**: Screenshot or share file links
- **Print**: Use browser print function (works best with simple version)

### Customization
- Colors can be modified in the CSS `<style>` section
- Content can be updated in the HTML body
- Emojis can be replaced with custom icons
- Layout can be adjusted using CSS Grid and Flexbox
