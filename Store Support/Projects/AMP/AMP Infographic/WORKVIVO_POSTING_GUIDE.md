# 📱 WorkVivo Social Media Post - Quick Guide

## How to Create the Image for WorkVivo

### Option 1: Browser Screenshot (Recommended)
1. **Open the file** `workvivo_social_post.html` in Chrome or Edge
2. **Press F11** to enter fullscreen mode (removes browser UI)
3. **Right-click** on the white post area
4. **Select** "Save image as..." or "Copy image"
5. **Save** as `winter-health-week41.png`
6. **Upload** directly to WorkVivo

### Option 2: Screenshot Tool
1. **Open** the HTML file in your browser
2. **Use** Windows Snipping Tool (Win + Shift + S)
3. **Capture** the entire white post area (1080x1920)
4. **Save** the screenshot
5. **Upload** to WorkVivo

### Option 3: Developer Tools
1. **Open** the file in Chrome
2. **Press** F12 to open DevTools
3. **Click** three dots menu → More tools → Network conditions
4. **Set** User Agent to "Mobile"
5. **Press** Ctrl+Shift+P
6. **Type** "screenshot" and select "Capture full size screenshot"
7. **Upload** the saved image to WorkVivo

## Image Specifications
- **Dimensions**: 1080 x 1920 pixels (9:16 vertical format)
- **Format**: PNG or JPG
- **Optimized for**: Mobile social media feeds
- **File size**: ~500KB-1MB (perfect for social posting)
