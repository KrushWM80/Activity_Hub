## APPENDIX: Data Quality Notes

### Data Collection Method
- Source: Core Work Project Tasks Tracker (SharePoint)
- Export Format: CSV from SharePoint list
- Data Completeness: 418 items with titles, 397 with completion dates, 21 in progress

### Known Limitations
1. No "Hours" field tracking in SharePoint list - unable to validate time allocation claims
2. "Status" field contains dates rather than status values - work is categorized by completion date
3. Some items may lack full categorization depending on field population

### Recommended Improvements
- Add explicit "Effort/Hours" field to tracker
- Add "Status" field with standardized values (Not Started, In Progress, Completed, Blocked)
- Ensure "Primary Work Type" is always populated
- Add "FY" field to better organize cross-fiscal-year work
