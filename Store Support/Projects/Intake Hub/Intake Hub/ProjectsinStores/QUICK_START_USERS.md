# Quick Start - Using the Auto-Login Dashboard

## For Everyone - First Time Access

### Option 1: You're on a Company Computer (Domain-Joined)

1. **Click this link**:
   ```
   http://weus42608431466.homeoffice.wal-mart.com:8001/index.html
   ```

2. **That's it!** 
   - You may see a quick notification: "Detecting your credentials..."
   - Then dashboard appears automatically
   - NO login form needed
   - NO username entry needed
   - NO password entry needed

3. **Dashboard is now open**
   - You'll see your projects and data
   - If you're an admin, you'll see the admin button

---

### Option 2: You're on a Remote Computer (Not Domain-Joined)

1. **Click the dashboard link**

2. **You'll see a login form**
   - It says "Detecting your network credentials..."
   - A spinner shows for a few seconds

3. **Enter your username**
   - Type your Walmart network username (the part before the @ in your email)
   - Example: If your email is `john.doe@walmart.com`, enter: `john.doe`
   - It auto-converts to lowercase (don't worry if you type capitals)

4. **Click "Sign In"** (or press Enter)
   - NO password field
   - NO password needed
   - Just your username

5. **Dashboard appears**
   - You're instantly logged in
   - All your data accessible

## Frequently Asked Questions

### Q: Why am I seeing a login form?
**A**: You're probably not on a company domain-joined computer. This is normal and expected for remote access. Just enter your username (no password needed).

### Q: Do I need a password?
**A**: NO! Passwords have been eliminated. You only need your username.

### Q: What if I see "Detecting credentials..."?
**A**: The system is checking if you're on a company computer. If you are (domain-joined), it'll auto-detect and log you in. If not, it'll show the username form after a few seconds.

### Q: Can I type my full email address?
**A**: No, just the username part. Example:
- ❌ Don't type: `john.doe@walmart.com`
- ✅ Do type: `john.doe`

### Q: What if I type the wrong username?
**A**: You'll be logged in, but might not see any data (if the username doesn't exist). You can try again by refreshing the page.

### Q: How long will I stay logged in?
**A**: You'll stay logged in for up to 8 hours. After 8 hours with no activity, you'll be logged out and need to log back in.

### Q: What if I'm on a company computer but it's asking for my username?
**A**: Your computer might not be properly configured for automatic detection. Just enter your username - the system will work fine.

### Q: Can I log out?
**A**: Yes, look for a logout button in the dashboard (usually top right). Your session will be cleared.

### Q: What if I see an error?
**A**: Check these:
1. **Server connection**: Is it at least showing a login form? If yes, server is working.
2. **Username spelling**: Make sure you typed it correctly (no spaces, no @domain)
3. **Try again**: Refresh the page and try once more
4. **Contact IT**: If still having issues, contact your IT support

## Troubleshooting

### Problem: "Connection failed - check server"

**Solution**:
- Make sure you have internet/network connection
- Try refreshing the page (F5 or Cmd+R)
- Check the URL is correct
- Wait a moment and try again

### Problem: Form keeps asking for username

**Solution**:
- Make sure you typed just the username (not email)
- No spaces or special characters
- Try clearing cookies: Settings → Privacy → Clear Browsing Data → Cookies
- Refresh page and try again

### Problem: I see "Login successful" but nothing happens

**Solution**:
- This is usually temporary - wait a few seconds
- If page doesn't load, try refreshing
- If still stuck, clear cookies and try again
- Check browser console (F12) for errors

### Problem: Dashboard loads but I see "No data" or wrong person's data

**Solution**:
- You might be logged in as a different user
- Try logging out and back in
- Make sure you typed the correct username
- Contact IT if the wrong user's data keeps showing

### Problem: I forgot my username

**Solution**:
- Your Walmart username is usually in your email address
- Example: If email is `john.smith@walmart.com`, username is `john.smith`
- If you don't know it, contact HR or your manager

## Advanced Users - Browser Tips

### Save Your Login (Optional)
Most browsers can remember that the site requires login with your username. You can enable browser password manager to autofill username (though password field won't appear).

### Clear Cookies If Having Issues
1. Press F12 (Developer Tools)
2. Go to Application/Storage
3. Find domain in list
4. Right-click → Clear All
5. Refresh page

### Check If Logged In
1. Open Developer Tools (F12)
2. Go to "Application" tab
3. Look for a cookie called `session_id`
4. If present, you're logged in
