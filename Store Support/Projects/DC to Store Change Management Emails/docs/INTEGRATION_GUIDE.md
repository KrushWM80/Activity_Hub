# INTEGRATION GUIDE: Adding Email Logging to Daily Tasks

## Overview
The dashboard is working with test data. To see **real emails** being logged automatically, you need to add the email logging hook to your existing email sending code.

## Quick Answer: Do You Need Integration?

**Short answer: NO - not yet!**

- ✅ Dashboard works standalone with test data
- ✅ You can test metrics and UI right now
- ✅ No changes needed to run the dashboard
- ⏳ Integration is optional for later when you want real emails

## When to Integrate

Integrate when you want:
- Real emails from `daily_check_smart.py` to be logged automatically
- Dashboard to show actual manager change data (not just samples)
- Email confirmations tracked from Outlook

## Integration: 2 Simple Steps

### Step 1: Add Import Statement
In `daily_check_smart.py`, add this near the top:

```python
from email_history_logger import log_email_send
```

### Step 2: Add Logging Call
After your email is sent successfully, add:

```python
# After successfully sending email to DC managers
log_email_send(
    dc_number=dc_number,           # e.g., 6020
    dc_type=dc_type,               # e.g., "Ambient" or "Perishable"
    recipients=recipient_list,      # List of emails sent to
    subject=email_subject,          # Email subject line
    changes={                       # Change counts
        'store_manager': count_store_mgr,
        'market_manager': count_market_mgr,
        'region_manager': count_region_mgr,
        'stores_affected': total_stores
    },
    test_mode=False,                # False for real emails
    status='sent'
)
```

### Example Location in daily_check_smart.py

```python
# ... existing email sending code ...

if send_email(recipient_list, subject, body_html):
    print(f"✓ Email sent to {dc_number}")
    
    # ADD THIS:
    log_email_send(
        dc_number=dc_number,
        dc_type=alignment_data['type'],
```

1. **Add integration** to `daily_check_smart.py` (Step 2 above)

3. **Run daily task**: Let hourly task send real emails
4. **Watch dashboard**: Metrics update automatically
