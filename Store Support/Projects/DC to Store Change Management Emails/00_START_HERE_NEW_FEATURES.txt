## 🔄 How They Work Together

### User Journey 1: Provide Feedback
```
1. User receives branded email
2. Clicks "Send Feedback" button
3. Email client opens with pre-addressed message
4. Sends to feedback@walmart.com
5. FeedbackHandler captures submission
6. Activity logged automatically
7. Admin sees in dashboard
8. Admin reviews and updates status
9. Full audit trail maintained
```

### User Journey 2: Find Store Managers
```
1. User receives branded email
2. Clicks "View Store Managers" button
3. Directed to manager directory
4. System extracts DC from email: 6020GM → DC 6020
5. Shows all 15+ managers for that DC
6. User clicks manager → email or call options
7. Direct contact with manager
8. Manager data auto-updates from system snapshots
```

### Admin Journey: Review Feedback
```
1. Admin accesses http://localhost:5000/admin
2. Sees dashboard with metrics: 15 total, 3 pending
3. Average rating: 4.2/5
4. Views recent feedback summary
5. Clicks feedback to open detail view
6. Reads full message + user metadata (DC, email, etc)
7. Updates status: new → reviewed
8. Adds admin notes: "Feature request acknowledged"
9. System logs change with admin name + timestamp
10. Activity log shows complete history
```

## 📊 Database Schema

### feedback_submissions
```sql
- id (INTEGER PRIMARY KEY)
- timestamp (DATETIME)
- user_email (TEXT)
- user_dc (TEXT) ← auto-extracted from email
- feedback_category (TEXT)
- rating (INTEGER 1-5)
- message (TEXT)
- status (TEXT: new, reviewed, resolved, archived)
- admin_notes (TEXT)
- submitted_via (TEXT: email, web, etc)
```

### feedback_activity_log
```sql
- id (INTEGER PRIMARY KEY)
- timestamp (DATETIME)
- event_type (TEXT: FEEDBACK_SUBMITTED, FEEDBACK_STATUS_CHANGED)
- feedback_id (FOREIGN KEY)
- admin_user (TEXT: who made the change)
- action (TEXT: description)
- details (TEXT: extra info)
```

## 🚀 Getting Started

### Minimal Setup (5 minutes)
```bash
# 1. Install Flask
pip install flask

# 2. Start admin server
python admin_app.py

# 3. Access dashboards
http://localhost:5000/admin                    # Admin dashboard
http://localhost:5000/store-manager-directory  # Manager directory
```

### Populate Manager Data
```python
from dc_to_stores_config import DCStoreManagerConfig

config = DCStoreManagerConfig()

# Add manually
config.add_manager("6020", {
    "name": "Sarah Johnson",
    "title": "Store Manager",
    "store_number": "1234",
    "email": "sarah.johnson@walmart.com",
    "phone": "555-0100",
    "market": "Dallas/Fort Worth"
})

# Or auto-update from snapshot
config.update_from_snapshot(your_snapshot_data)
```

### Integrate Feedback
```python
from feedback_handler import FeedbackHandler

handler = FeedbackHandler()

# Submit feedback
handler.submit_feedback(
    user_email="6020GM@email.wal-mart.com",
    feedback_category="UI",
    rating=4,
    message="Great tool!"
)

# Get stats
stats = handler.get_feedback_stats()
print(stats['average_rating'])  # 4.0
```
